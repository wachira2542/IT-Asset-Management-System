สแกน QR Code อุปกรณ์ไอที

สแกนด้วยกล้องจริง (Webcam / Mobile Camera)

หันกล้องไปสแกนป้าย QR Code ที่ติดบนอุปกรณ์คอมพิวเตอร์เพื่อเปิดแบบฟอร์ม

เครื่องมือจำลองการสแกน (Scanner Simulator)

สำหรับทดลองสแกนบนเครื่องที่ไม่มีกล้องเว็บแคม หรือเพื่อการนำเสนอ

คำแนะนำในการทำงาน:

  • สแกนเครื่องสถานะ Available (ยืมชั่วคราว - ว่าง) -> จะเปิดหน้าทำรายการ "ขอยืมคอมพิวเตอร์" โดยอัตโนมัติ
  • สแกนเครื่องสถานะ Rented (กำลังยืมอยู่) -> จะเปิดหน้าทำรายการ "ส่งคืนคอมพิวเตอร์" โดยอัตโนมัติ
  • สแกนเครื่องที่ถูกจ่ายถาวร (Deployment) หรือสถานะอื่นๆ -> จะแสดงประวัติการใช้งานและรายละเอียดของอุปกรณ์เครื่องนั้นๆ

ประวัติการทำรายการยืม-คืนเครื่องทั้งหมด

วันเวลาทำรายการ Asset Tag / รุ่น พนักงานผู้ยืม (แผนก) ผู้บันทึกยืม กำหนดส่งคืน วันที่คืนจริง ผู้รับคืน สภาพตัวเครื่อง
